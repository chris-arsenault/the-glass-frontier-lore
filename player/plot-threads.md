# Plot Threads

Overarching narrative arcs that span the full timeline. Each is a long causal chain with minimal branching — a spine that other entities attach to. Contains DM knowledge where noted. Contains future state (things that haven't happened yet but will, given the trajectory).

These are the macro stories. Individual entries are episodes within them.

---

## Thread 1: The Hidden War [DM]

*The elves prepare for an enemy nobody else knows exists.*

```
Ring Age: Elves build the ring, understand resonance at deep level
  → 2240: The Adversary destroys the ring
  → 2240-2280: Elves decide to disappear, scrub their history, retreat to hidden habs
  → 2240-2369: 130 years of preparation — weapons, strategy, training
  → 2355: The Bloom tears reality → Adversary begins seeping through
  → 2358+: Elves increase presence in Bloom Zones, cooperate with Displacement Council
  → Present: Dern Talish cooperates with elves, Adversary slowly expanding
  → Future: [The return — the war the elves have been preparing for]
```

**Themes:** Personal Sacrifice to the Greater Good, The Cost of Absence, The Price of Knowledge

**DM notes:** This thread is invisible to players until discovery. The entry points are: Echo River fragments, the Redaction pattern, Dern Talish's behavior, duskgrain's true nature, elven operatives in the general population.

## Thread 2: The Bloom Containment

*An industrial accident becomes the defining crisis of a generation.*

```
Rekindling: Coremark pioneers aggressive Shear salvage without understanding resonance ecology
  → Contested Reach: Coremark expands into mid-drift, overloads Pyre refinery
  → 2355: Pyre overload triggers the Silent Bloom — resonance cascade tears reality
  → 2355-2358: Emergency coalitions form (Bloom Coalition), Lira Vashtenri develops Lira's Wall
  → 2358: Verge Compact established, containment stabilizes
  → 2358-2362: Coalition fractures, Displacement Council forms from refugees
  → 2362-2365: Bitter Reach — second wave of conflicts
  → 2365: Tempered Accord ratified, partly because Bloom cooperation proved it could work
  → Present: Bloom Zones still contained but expanding, Council stretched thin
  → Future: [Containment fails somewhere? New Bloom event? Coremark triggers a second cascade?]
```

**Themes:** Cooperation Against the Odds, The Cost of Absence (the Bloom happened because elven knowledge was gone), Identity Through Adversity (the Displacement Council, the Accord)

## Thread 3: The Reconnection

*A fractured civilization remembers it was once whole.*

```
2240: The Glassfall severs everything
  → 2240-2305: Signal Famine — 65 years of isolation, cultural divergence
  → 2305: Glasswake reconnects with Hab Meridian (accidental, via salvaged transmitter)
  → 2305+: Span Nine rediscovered — Sithari connects to Pelhari and Threshold Station
  → 2310: The Absence confirmed — the elves are gone everywhere
  → 2310-2330: Ring hab contact — cultural shock, some habs refuse (shuttered habs)
  → 2320+: Shear salvage begins, Vantara founded, trade networks emerge
  → 2330+: The Keel established, Kite Guild logistics
  → 2340-2355: Contested Reach — the cost of reconnection (disputes, corridor conflicts)
  → 2365: Tempered Accord — the framework for coexistence
  → Present: System reconnected but not unified, far-drift habs still being discovered
  → Future: [New habs contacted that challenge the Accord's assumptions]
```

**Themes:** Identity Through Adversity, Preservation vs Progress, Cooperation Against the Odds

## Thread 5: The Question of Governance

*How do you build shared structures for a civilization that spent sixty years becoming strangers?*

```
Ring Age: Unified governance under ring infrastructure
  → 2240: Everything breaks — governance fragments
  → 2240-2305: Each community develops its own system (Fermata's jazz, Sithari's Continuity, shuttered habs' isolationism)
  → 2305-2340: Rekindling — communities discover incompatible systems
  → 2340-2355: Contested Reach — some conflicts are about resources, some are about who decides
  → 2355-2362: Bloom forces cooperation — proof that shared structures can work
  → 2358: Verge Compact — first system-wide regulatory body (salvage-specific)
  → 2365: Tempered Accord — the general framework (mediation, not government)
  → Present: The Accord holds but strains. Sithari's influence is resented. Far-drift habs don't fit.
  → Future: [Accord crisis? New governance model? The Accord fails and something better emerges?]
```

**Themes:** Preservation vs Progress, Cooperation Against the Odds, Identity Through Adversity

## How to Use These

When writing a new entity, check which plot thread(s) it connects to. An entity that doesn't touch any thread is either:
1. A leaf node (local color, fine if it's minor)
2. Disconnected from the narrative (a problem if it's prominent)
3. The start of a new thread (worth discussing)

When writing a pivotal moment, check which thematic cores it resonates with. A moment that doesn't echo any theme is either:
1. Texture (fine)
2. Tonally adrift (worth revising)

Plot threads and thematic cores are not rules. They are gravity — things orbit around them naturally, and when something drifts too far, it's worth asking why.
