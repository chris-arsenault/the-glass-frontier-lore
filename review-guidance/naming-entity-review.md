# Entity Naming Review — Specific Checks

Targeted review criteria for each entity type. Run through these when adding or auditing entity names.

## Location Names

Locations are named by the people who live there or found them. Ask:
- **Who found this place?** A Sithari surveyor names differently than a Hab-Worlder salvage crew.
- **When?** Rekindling-era names should feel improvised and functional. Accord-era names might be more deliberate.
- **Has the name changed?** Many locations have an official name and a name everyone actually uses. Include both where relevant.
- **Is it named after someone?** Real places often are. "Verathi Station" is better than "The Arbitration Station."

Common fails:
- Every location named for its defining feature ("Bloom Zones," "Echo Rivers")
- No mundane location names (every location sounds significant)
- All locations using the same naming register

## Event Names

Events are named after the fact, usually by historians or by popular usage. Ask:
- **What did people call it while it was happening?** Often different from the historical name.
- **Who named it?** Sithari academics name events differently than Hab-Worlder survivors.
- **Is the name earned?** "The Glassfall" is a good earned name — evocative, specific, sticky. Not every event needs this treatment. Some can just be "the 2345 corridor dispute."

Common fails:
- Every event capitalized and formalized ("The Shardfall Incident")
- All events named using the same "The [Noun] [Noun]" pattern
- No events referred to by their date or location instead of a coined name

## Faction Names

Factions name themselves (or get named by others). Ask:
- **Did they choose this name?** Self-chosen names signal aspiration. Names given by others signal reputation.
- **Does it sound like something a committee would produce?** Some factions ARE committees and should sound like it. Others should sound like what people actually call them.
- **Is there a short form?** "The Verge Compact" → "the Compact." "Prismwell Kite Guild" → "the Guild" or "Prismwell."

Common fails:
- Every faction name follows [Adjective/Place] [Role] [Organization Type]
- No factions with names that don't explain their function
- All factions sound equally formal

## Artifact Names

Artifacts are named by their makers, their users, or legend. Ask:
- **Is this a technical designation or a popular name?** "The Damper Array" is technical. A popular name might be "Lira's Wall" or just "the Array."
- **Does the name tell you too much?** An artifact named "The Resonance Containment Device" is a spec sheet. One named "the Chorus" is a mystery.
- **Is it named after its effect, its maker, or something unrelated?** Mix these.

Common fails:
- Every artifact name describes its function
- No artifacts with names that have been corrupted, shortened, or misattributed over time

## People Names

See [naming-review.md](naming-review.md) and [naming-conventions.md](../player/concepts/naming-conventions.md) for the cultural/species conventions. Additional checks:
- **Does the name match the character's culture and species?** Cross-check against naming conventions.
- **If they code-switch, is that reflected?** An orc with a Sithari professional name and an orcish mononym should have both referenced.
- **Are there nicknames or shortened forms?** Real people have these. "Lira Vashtenri" → "Lira Vasht" → "Vasht" depending on who's talking.
